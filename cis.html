    Express.Js

    Express.js is a Node.js framework (Pre-built Node.js file) used to make a connection to a server. It's main purpose is to help handle numerious different http requests at once. Using different methods to send or post information, and to retrieve or get. Using express is helpful for smaller scale projects to test without hurting a server or risking data. Being able to handle numerious http requests, express works well with API's (Aplication Protocol Internface). By just being able to handle so many request for information and authentification.

    var express = require('express');
    var app = express();

    app.set('view engine', 'ejs');

    // index page
    app.get('/', function(req, res){
    res.render('pages/index');
    });

    // cis page
    app.get('/cis', function(req, res){
    res.render('pages/cis');
    });

    app.listen(8080);

    In this example I am using express to create a local server. By first using the require method, the express frame work gets brought in and stored into a variable named "app". In this project instead of having an HTML, an EJS(Embeded JavaScript) file is the index asigned as the view engine. EJS is an html file that can render express JavaScript within the page. The reason for using this file extension it to allow express to switch between files with little to no package loss. The app.get method is how this is possible by retrieveing from the project folder.


    Bootstrap

    Bootstrap is a CSS(cascading style sheet) framework that is used to help create more clean and efficient code. The main use of hte framework is used to help add stylish properties onto elements. A key feature is the positioning layout, and using less code to make more happen. Helps to creates grids,and basic styling such as in Microsoft applications.
